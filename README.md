# Esoteric Ebb 简体中文汉化补丁

[Esoteric Ebb](https://store.steampowered.com/app/2057760/Esoteric_Ebb/) 的社区简体中文汉化补丁。当前版本：**v1.2**

## 特性

- **75,000+ 条文本翻译**，覆盖对话、任务、角色创建、UI 界面
- **286 个 Ink 故事文件**完整翻译，涵盖全部游戏剧情对话
- **角色创建界面**中文化 — 六大属性、子职业、背景专长
- **主菜单 / 设置 / 存档 / 读档界面**翻译
- **IL2CPP 属性名称修补** — 属性加成标签（+3 力量、+1 人格等）原生显示中文

## 安装方法

1. 在 [Releases](../../releases) 页面下载最新版本的 zip 压缩包
2. 将压缩包内容**解压到游戏安装目录**：
   ```
   ..\Steam\steamapps\common\Esoteric Ebb\
   ```
3. 出现覆盖提示时选择**全部覆盖**
4. 启动游戏即可看到中文界面

## 卸载方法

在 Steam 中：**右键游戏 → 属性 → 已安装文件 → 验证游戏文件完整性**

这会将所有修改过的文件恢复为原始状态。

## 已知问题

- 字体图集仅包含约 3,752 个 CJK 汉字，缺少约 415 个常用字（如：嗯、呃、挺、尴尬、骷髅、铠 等），部分翻译使用了替代用词
- 角色创建界面的属性详细描述文本仍为英文
- 游戏内场景 UI 文本暂未翻译
- 部分制作名单 / 版权声明保留英文

## 技术信息

本补丁使用直接二进制修补方案，无需任何第三方运行时。仅替换游戏数据文件，不修改可执行文件。
