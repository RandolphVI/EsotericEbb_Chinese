# Esoteric Ebb 简体中文汉化补丁

[Esoteric Ebb](https://store.steampowered.com/app/2057760/Esoteric_Ebb/) 的社区简体中文汉化补丁。当前版本：**v2.0**

## 特性

- **73,900+ 条文本翻译**，覆盖对话、任务、角色创建、UI 界面
- **286 个 Ink 故事文件**完整翻译，涵盖全部游戏剧情对话
- **主菜单、角色创建界面**中文化 — 六大属性、子职业、背景专长
- **游戏内 UI 全中文化** — 法术书、物品栏、属性面板、任务日志等
- **词汇表术语高亮**（680+ 条术语，点击可查看说明）
- **BepInEx 翻译插件**，解决动态 UI 文本、法术描述等运行时翻译
- **字体图集升级** — 大幅改善可读性

## 安装方法

1. 在 [Releases](../../releases) 页面下载最新版本的 zip 压缩包
2. 将压缩包内容**解压到游戏安装目录**（选择全部覆盖）：
   ```
   ..\Steam\steamapps\common\Esoteric Ebb\
   ```
3. 确认游戏根目录中存在 `winhttp.dll` 和 `doorstop_config.ini`（BepInEx 加载器）
4. 启动游戏即可看到中文界面

## 卸载方法

在 Steam 中：**右键游戏 → 属性 → 已安装文件 → 验证游戏文件完整性**

这会将所有修改过的游戏数据文件恢复为原始状态。如需彻底移除 BepInEx，还需手动删除游戏根目录中的 `winhttp.dll`、`doorstop_config.ini`、`.doorstop_version`、`BepInEx/` 文件夹和 `dotnet/` 文件夹。
