# Esoteric Ebb 简体中文汉化补丁

[Esoteric Ebb](https://store.steampowered.com/app/2057760/Esoteric_Ebb/) 的社区简体中文汉化补丁。当前版本：**v1.6**

## 特性

- **75,000+ 条文本翻译**，覆盖对话、任务、角色创建、UI 界面
- **286 个 Ink 故事文件**完整翻译，涵盖全部游戏剧情对话
- **主菜单、角色创建界面**中文化 — 六大属性、子职业、背景专长
- **游戏内 UI 全中文化** — 法术书、物品栏、属性面板、任务日志等（持续进行中...）
- **字体图集扩充**

## 安装方法

1. 在 [Releases](../../releases) 页面下载最新版本的 zip 压缩包
2. 将压缩包内容**解压到游戏安装目录**：
   ```
   ..\Steam\steamapps\common\Esoteric Ebb\
   ```
3. 出现覆盖提示时选择**全部覆盖**
4. 启动游戏即可看到中文界面

## 卸载方法

在 Steam 中：**右键游戏 → 属性 → 已安装文件 → 验证游戏文件完整性**

这会将所有修改过的文件恢复为原始状态。
