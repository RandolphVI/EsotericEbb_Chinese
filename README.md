# Esoteric Ebb 简体中文汉化补丁

[Esoteric Ebb](https://store.steampowered.com/app/2057760/Esoteric_Ebb/) 的社区简体中文汉化补丁。当前版本：**v1.0**

## 特性

- **75,000+ 条文本翻译**，覆盖对话、任务、角色创建、UI 界面
- **286 个 Ink 故事文件**完整翻译，涵盖全部游戏剧情对话
- **角色创建界面**全中文化 — 六大属性描述、子职业、16 个背景专长
- **设置 / 存档 / 读档界面**完整翻译
- **IL2CPP 属性名称修补** — 属性加成标签（+3 力量、+1 人格等）原生显示中文

## 安装方法

1. 在 [Releases](../../releases) 页面下载最新版本的 zip 压缩包
2. 将压缩包内容**解压到游戏安装目录**：
   ```
   ..\Steam\steamapps\common\Esoteric Ebb\
   ```
3. 出现覆盖提示时选择**全部覆盖**
4. 启动游戏即可看到中文界面

## 卸载方法

在 Steam 中：**右键游戏 → 属性 → 已安装文件 → 验证游戏文件完整性**

这会将所有修改过的文件恢复为原始状态。

## 兼容性

- **游戏版本**：基于 2026 年 3 月 Steam 版本制作
- **系统要求**：Windows 10/11
- **游戏更新**：如果游戏更新，资产文件可能被覆盖，需重新安装补丁

## 已知问题

- 字体图集缺少"魅"字符，Charisma 翻译为"人格"而非"魅力"
- 部分制作名单 / 版权声明保留英文
- 游戏内少数 UI 文本可能仍为英文（持续更新中）
