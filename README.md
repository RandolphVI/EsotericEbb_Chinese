# Esoteric Ebb 简体中文汉化补丁

[Esoteric Ebb](https://store.steampowered.com/app/2057760/Esoteric_Ebb/) 的社区简体中文汉化补丁。当前版本：**v1.6**

## 特性

- **74,000+ 条对话翻译**，通过 Dialogs 本地化表 + Ink 故事文件双轨全覆盖
- **286 个 Ink 故事文件**翻译，涵盖全部游戏剧情对话
- **角色创建界面**中文化 — 六大属性、背景专长、预设角色等
- **字体图集扩充** — 4,405 个字形覆盖 4,290 个 CJK 字符（48px 高清渲染）
- **术语统一** — 严格遵循 D&D 5e 中文术语表

## 安装方法

1. 在 [Releases](../../releases) 页面下载最新版本的 zip 压缩包
2. 将压缩包内容**解压到游戏安装目录**：
   ```
   ..\Steam\steamapps\common\Esoteric Ebb\
   ```
3. 出现覆盖提示时选择**全部覆盖**
4. 启动游戏即可看到中文界面

## 卸载方法

在 Steam 中：**右键游戏 → 属性 → 已安装文件 → 验证游戏文件完整性**

这会将所有修改过的文件恢复为原始状态。

## 兼容性

- **游戏版本**：基于 2026 年 4 月 Steam 版本 (**Patch v1.2 / v1.2.1**) 制作
- **系统要求**：Windows 10/11
- **游戏更新**：如果游戏更新，资产文件可能被覆盖，需重新安装补丁

## 已知问题

- 背景专长卡片的属性加成标签（如 "+2 Charisma"）仍为英文，该文本由游戏编译代码生成，暂无法通过资产修补翻译

## 技术信息

本补丁使用直接二进制修补方案，无需任何第三方运行时。仅替换游戏数据文件，不修改可执行文件。
